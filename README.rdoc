== README

Update 
============
Cette web app est déployée sur Heroku: https://glacial-everglades-3548.herokuapp.com/

Installation
============


    git clone git://github.com/jneree/eChat.git
    cd eChat
    bundle install
    rake db:migrate

Démarrer le serveur local:

    bundle exec rails s

Vous pouvez maintenant avoir accès à l'app via 127.0.0.1:3000 dans le navigateur.
